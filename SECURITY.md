# Security

Dominus OS takes security seriously. This repository contains public documentation only — no source code, credentials, or internal schemas.

## Reporting Security Issues

If you discover a security concern related to Dominus OS or Dominus Foundry, please report it responsibly:

- **Email:** security@dominusfoundry.com
- **Response time:** We aim to acknowledge reports within 48 hours.

Please do not open public issues for security concerns.
